<h1>Summary</h1>

<p>Creates a link to feed for use in the header portion of an xhtml page.</p>

<h2>&lt;txp:mkp_section_header_link /&gt;</h2>

<h2>Attributes</h2>

<h3>flavor: rss (default), atom</h3>

<h3>section: NULL (default), [site section]</h3>

<h2>Example</h2>

<p><span style="padding: 10px; border: 1px solid #999; background: #eee; display: block; margin: 10px 0 0 0; text-indent: 0; text-align: left; color: #600;width:600px;"><code><txp:mkp_section_header_link flavor="atom" section="article" /></code></span><br /></p>
